# Canvas des 7 Fenêtres — Concevoir un projet de formation

Outil pédagogique interactif (autonome, un seul fichier HTML) pour concevoir un projet de formation en santé selon le modèle des **7 Windows of Instructional Design**, avec accent **ludopédagogie**.

## Deux onglets
1. **Raisonnement pédagogique** — en-tête de cadrage + 7 fenêtres (Analyse → Évaluation) à menus déroulants.
2. **Scénario & programme** — séquenceur par blocs de ~90 min, formats 1 h → 3 jours, intersession d'ancrage, programme horodaté.

## Exports
TXT · Word (.doc) · Excel (.xls) · HTML (→ PDF via Ctrl/Cmd+P) · Sauvegarde/Reprise de session (.json).
